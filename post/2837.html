《模仿游戏 The Imitation Game》观后感

导演: 莫滕·泰杜姆
编剧: 格拉汉姆·摩尔 / 安德鲁·霍奇斯
主演: 本尼迪克特·康伯巴奇 / 凯拉·奈特莉
类型: 剧情 / 同性 / 传记 / 战争
制片国家/地区: 英国 / 美国
语言: 英语 / 德语
上映日期: 2015-07-21(中国大陆) / 2014-11-14(英国) / 2014-12-25(美国)
片长: 114分钟
又名: 解码游戏(港) / 模拟游戏

这部电影根据传奇密码学家艾伦·图灵（Alan Turing）的真实生活故事改编，影片描述了在第二次世界大战最黑暗的日子里，图灵及其团队在英国政府最高机密的布莱夫利公园破解德国纳粹的通讯密码，帮助盟军赢得二战胜利的故事。

电影分为3条时间线：1928年，图灵的学生时代；1939年-1945年，图灵在二战时期的工作；1951年，图灵的家遇到抢劫，并接受警察的调查。

而我写这篇文章，不是因为图灵的“计算机科学之父”的身份，也不是因为第二次世界大战，更不是因为影片对同性恋的态度。而是通过影片的演绎，演员的表演，片中的阿兰图灵体现出了非常典型的孤独症症状。

孤独症，又称为自闭症，是一种疾病，而不是精神疾病。因为我的儿子在儿童医院做儿保的时候被怀疑有孤独症的倾向，因此我一直非常关注孤独症的各种症状，并仔细的观察儿子的变化。

影片中的阿兰图灵不善于人际交往，只能理解语言中的表面含义，说话直来直去，其实，跟《生活大爆炸》中的谢尔顿相同，是典型的孤独症症状。但是他们都能进行正常的日常生活，并且在自己的领域里达到很高的成就，按照图灵的话来说，他们只是思维方式跟大部分人不同而已，不能说他们就是病态的。也许影片中的这句台词只是针对同性恋这个问题而已，但是对于自闭症这个群体来说也是一样。

图灵学生时代的好友向他介绍密码学的时候说，密码学就是把想要告诉别人的信息加密，接收方通过解密之后才能看到信息，图灵回答道，生活中说的每句话对于他来说都是加密的，因为我们说的话，很多都不是我们的真实想法，而是拐弯抹角的，他不能理解这种拐弯抹角的语言想要表达的对方的真实想法。因此他的好友会告诉他很适合学习密码学。

就像电影中他的同事叫他去吃饭的时候的对话，他的同事反复给他强调：我们要去吃午饭了！但是阿兰不能理解这句话中的‘要不要一起吃午饭’的真实含义，因此一直不理睬同事，直到有人问‘谁饿了？’才回答他也饿了。这种‘不合群’的为人处事方式，很容易在刚刚接触到的陌生人面前造成一种‘装怪’、‘不好接触’的映像，因此，有孤独症的人很难有朋友。但如果经过长期接触，就会发现其实他们也就是说话直接，你也要这样直来直去的对他说话而已。

而阿兰图灵的团队中其他的密码学家，起初也从视他为怪物、怪胎，变成了他的好朋友，也就像生活大爆炸里谢尔顿的好友莱纳德一样。

也许，国外的演员们在饰演这种孤僻、怪异的角色之前都看过有关于孤独症的书，在饰演角色的时候，都参考过孤独症的一些症状，使他们看起来确实与常人不一致，然后又有科学的依据来对这种怪异的行为进行合理的解释。

而我的儿子快3岁了，他的手脚精细运动，语言发育等等还是有点落后于同龄儿童，虽然确定没有很严重的孤独症，但是我现在已经没有当初那么担心了，就算他有一点孤独症的倾向，也会是轻微的。而我作为父亲，不能也不会把他当成一个特殊的存在去对待，每一个孩子都是独一无二的。
